**Умный автоответчик для Telegram** — локальное приложение, которое автоматически отвечает на первое сообщение от новых пользователей.

## ✨ О проекте

WinAutoReply — это простой и безопасный автоответчик для Telegram, работающий полностью локально. Программа создана для тех, кто ценит приватность и хочет автоматизировать общение без передачи данных на внешние серверы.

## 🔎 Основные возможности

- Отвечает **только на первое сообщение** от каждого пользователя (чтобы не спамить).
- Автоматический вход в Telegram-аккаунт.
- Полностью локальная работа: **данные не уходят на сервер**.
- Простой запуск — достаточно запустить скрипт или исполняемый файл.
- Автоматическая установка Python (если не установлен).
- Поддержка **Windows** и **Linux**.

## 🛠 Технологии

- **Python 3** — язык программирования.
- **Telethon** — библиотека для работы с Telegram API.
- **PyInstaller** — для создания исполняемых файлов.

## 🔒 Безопасность и приватность

Мы серьёзно относимся к безопасности ваших данных:

- ✅ **Ничего не отправляется на внешние серверы** — всё происходит на вашем компьютере.
- ✅ **Все данные хранятся локально** в папке `Документы/WinAutoReply`.
- ✅ **Открытый исходный код** — вы можете проверить код и убедиться в его безопасности.
- ✅ **Сессии Telegram** хранятся только у вас — никто не имеет к ним доступа.

## 🚀 Запуск

### Вариант 1: Через Python (рекомендуется для разработчиков)

1. Убедитесь, что у вас установлен **Python 3.8+**.
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
Запустите приложение:

bash
python3 app.py
Вариант 2: Скомпилированная версия (для пользователей без Python)
Windows: дважды кликните по файлу build.bat.

Linux: выполните в терминале:

bash
./build_linux.sh
После запуска следуйте инструкциям в консоли для авторизации в Telegram.

⚙️ Настройка
Все настройки хранятся в папке config/. Вы можете изменить:

Текст автоответа.

Задержку перед ответом.

Фильтрацию сообщений (например, игнорировать ботов).

Подробнее смотрите в файле config/config.json и комментариях внутри него.

❓ Часто задаваемые вопросы (FAQ)

Q: Нужно ли устанавливать Python?
A: Нет, если вы используете скомпилированную версию (build.bat или build_linux.sh). Скрипт установит Python автоматически.

Q: Сохранится ли сессия Telegram после перезапуска?
A: Да, сессия сохраняется в папке sessions/. Вам не нужно авторизовываться заново.

Q: Можно ли настроить ответ для конкретных пользователей?
A: В текущей версии автоответчик отвечает всем новым пользователям. В будущих обновлениях планируется расширенная фильтрация.
